Media Production

Generate video content, 3D assets, and rich media from natural language prompts. AI OS integrates with Remotion for programmatic video, Google Vids for collaborative editing, and Blender MCP for 3D scene generation.

Remotion Video

Remotion enables programmatic video creation using React components. The Media Producer agent translates natural language prompts into Remotion compositions:

  • Script generation — AI writes the video script, scene breakdown, and timing
  • Component assembly — Automatically selects and configures Remotion components (text overlays, transitions, animations)
  • Rendering — Server-side rendering to MP4 with configurable resolution and frame rate
  • Batch production — Generate multiple video variants for A/B testing

Google Vids

For collaborative video editing, AI OS integrates with Google Vids:

  • AI-generated storyboards pushed to Google Vids for human review
  • Collaborative editing with team members in real time
  • Automatic captioning and transcript generation
  • Export to multiple formats and platforms

Blender 3D

The Blender 3D agent creates three-dimensional scenes and product renders from text descriptions via the Blender MCP integration:

  • Scene generation — Describe a scene and get a complete Blender project file
  • Product renders — Studio-quality product photography from 3D models
  • Environment design — Interior/exterior scenes for architectural visualization
  • Animation — Simple object animations (rotations, camera paths, material transitions)

Vibe Design Studio

The Vibe Designer agent generates UI mockups and interaction designs from prompts:

  • Prompt-to-UI — Describe a screen and get a visual mockup
  • Predictive heat maps — AI-estimated attention maps showing where users will look first
  • Interaction flows — Multi-screen user journey maps
  • Component extraction — Break mockups into reusable component specs

Dashboard Integration

The Media view in the dashboard provides:

  • Natural language prompt input with type selector (video, 3D, UI)
  • Real-time progress tracking via WebSocket
  • Preview panel for generated media
  • History of all generated assets with re-generation options
  • Batch queue integration for high-volume production
