Under the Hood

6-Layer Agentic Architecture

Every layer is modular — swap models, add agents, chain skills, all without touching core code.

01

Knowledge & Memory

Vault system with raw intake, synthesized wiki, and semantic search. Session hooks auto-load context.

02

Protocol & Identity

Three-file soul stack: immutable guardrails, operator preferences, agent personas. Per-project context overrides.

03

UI/UX Generation

DESIGN.md dual-structure tokens, Vibe Design prompt-to-screens, WCAG linter, brand clone from URL.

04

Multimodal Media

Remotion video, Google Vids, Blender 3D, batch generation — all template-driven with cost tracking.

05

Execution & Automation

Multi-model routing (Opus/Sonnet/Haiku/DeepSeek/Grok), skill chaining, CRON loops, Playwright browser agent.

06

Output & Monetization

Product factory, lead pipeline, marketing hub, Golden Loop — autonomous revenue generation.
