# Agent Roster Knowledge Base

## Tier 1 — Strategic (Opus)
- **orchestrator**: Master coordinator, task routing, team assembly
- **architect**: System design, technical planning
- **reviewer**: Critical analysis, read-only veto power
- **security-auditor**: Vulnerability scanning, dependency auditing

## Tier 2 — Professional (Sonnet)
- **researcher**: External information gathering with citations
- **coder**: Implementation from specifications
- **writer**: Documentation and content creation
- **data-wrangler**: Data processing and analysis
- **research-architect**: Research methodology design
- **synthesis**: Cross-source pattern identification
- **report-compiler**: Final deliverable assembly

## Tier 3 — Scout (Haiku)
- **scout**: Lightweight intelligence gatherer, tech radar sweeps

## Tier 4 — Economy (DeepSeek V4)
- **deepseek-worker**: Bulk content, batch SEO, data transformation

## Support Agents
- **qa**: Testing and verification (Sonnet)
- **safety**: Guardrail enforcement (Opus)
- **factory**: Agent file generation (Sonnet)
