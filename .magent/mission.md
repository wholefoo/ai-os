# Mission: AI OS Orchestration Lab

## Objective
Build and operate a multi-agentic AI Operating System that transforms Claude Code into an accessible platform for orchestrating complex workflows through a visual dashboard.

## Success Criteria
- Non-technical users can trigger workflows via the dashboard
- Agent team can execute end-to-end tasks autonomously
- All actions are auditable via decision logs
- System handles research, content creation, data processing, and code generation

## Constraints
- File-based architecture (no external databases required)
- Must run on a standard VPS (4GB RAM minimum)
- Human-in-the-loop for irreversible actions
- All outputs land in .magent/artifacts/
